Pelayanan kami

Hosting minecraft

Di RexxCloudID kami menawarkan hosting server minecraft yang kuat dan dapat disesuaikan sepenuhnya dengan software untuk Bedrock,PocketMine-MP,Paper, Fabric, dan Forge.Dll. PocketMine-MP adalah aplikasi simpel tapi bermanfaat bagi pemain Minecraft. Edisi ini diinstal di perangkat Android mereka agar mereka dapat cepat dan mudah membuat server pribadi yang berfungsi penuh dan dapat dimodifikasi. Paper memungkinkan pengalaman server berkinerja tinggi, sementara Fabric dan Forge memungkinkan pemain menambahkan mod dan add-on untuk memperluas dan mempersonalisasi pengalaman Minecraft mereka. Dengan akses penuh ke data server Anda dan kemampuan untuk mengunggah plugin dan mengelola file, RexxCloudID memungkinkan Anda menciptakan pengalaman bermain Minecraft terbaik. Antarmuka web intuitif kami menyederhanakan manajemen server sehingga Anda dapat fokus pada hal yang benar-benar penting - pengalaman bermain game Anda.

  • Bedrock
  • PocketMine-MP
  • Forge
  • Fabric
  • Paper

Teamspeak Server

Di RexxCloudID, kami menawarkan server TeamSpeak yang tangguh dengan hingga 32 slot, menyediakan platform sempurna untuk komunikasi lancar dalam komunitas game Anda. Dengan Endelon Hosting Anda akan merasakan komunikasi suara kelas satu yang dipadukan dengan fungsi manajemen ekstensif.
